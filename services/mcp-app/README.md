# MCP App

Owns the FastMCP interface exposed to AI clients.

## Responsibilities

- MCP tools for searching and inspecting coupons.
- MCP resources for merchant, offer, and metadata views.
- Prompts or client-facing guidance related to coupon search.
- Query-side access to normalized coupon data.

## Non-responsibilities

- Provider feed ingestion.
- Affiliate network authentication.
- ETL scheduling or freshness jobs.
